# Resource Monitor

Ferramenta para profiling, análise de namespaces e gerenciamento de cgroups em sistemas Linux, desenvolvida em C/C++ seguindo as especificações acadêmicas.

## Estrutura do Projeto

- `src/`: Código-fonte dos módulos principais
- `include/`: Headers
- `docs/`: Documentação
- `tests/`: Testes unitários
- `scripts/`: Scripts auxiliares

## Compilação

```sh
make
```

## Uso

Veja detalhes de cada ferramenta e exemplos de uso no README final.

## Dependências

- Compilador C/C++ (gcc/g++)
- Linux (acesso a /proc, /sys/fs/cgroup)
